ColorPallet

  • 컬러 파레트
  • jquery, Bootstrap 없이도 동작 가능
  • http://www.mins01.com/mh/tech/read/1218
  • https://github.com/mins01/ui_ColorPallet/
  • https://mins01.github.io/ui_ColorPallet/

create

  • var cp:html_node = ColorPallet(opt);
    @param Object opt {defColor:기본색,bookmark:즐겨찾기 기본 배열,"maxHistory":최대 히스토리 수,"localStorageHistoryKey":히스토리용 키}
    @return cp:html_node tag div
  • ex
    var cp = ColorPallet({bookmark:["#abc","#369"],"maxHistory":10,"localStorageHistoryKey":"save_key"});
    InputRangeBox.autoInit(cp); //옵션
    document.body.appendChild(cp); //옵션

method

  • colorData:Object = cp.getData() 컬러 데이터 가져오기
  • cp.setData(colorData:Object) 컬러 데이터 설정하기
  • stringHSL:String = cp.toStringHSL([colorData:Object]) 색을 hsl(111,99%,99%) 로 나타냄
  • stringRGB:String = cp.toStringRGB([colorData:Object]) 색을 rgb(111,222,99) 로 나타냄
  • stringHEX:String = cp.toStringHEX([colorData:Object]) 색을 #aabbcc 로 나타냄
  • cp.setHEX(HEX:String) HEX로 값을 설정
  • cp.setHSL(h:Number,s:Number,l:Number)hsl 값으로 설정
  • cp.setRGB(r:Number,g:Number,b:Number)rgb 값으로 설정
  • cp.setColorString(colorString:String) colorString 값을 설정
  • stringJSON:String = cp.toString() 컬러 데이터를 JSON 문자열로(ex:{"h":0,"s":72,"l":26,"r":115,"g":19,"b":19}) 출력
  • colorData:Object = cp.parseColorString(colorString:String) 컬러 문자열을 컬레 데이터로 변환
  • cp.confirm() confirm 동작. confirm 이벤트가 발생됨
  • cp.cancel() cancel 동작. cancel 이벤트가 발생됨

참고

  • colorData:Object 모양 {"h":0,"s":72,"l":26,"r":115,"g":19,"b":19}
  • stringHSL:String 모양 hsl(111,99%,99%)
  • stringRGB:String 모양 rgb(111,222,99)
  • stringHEX:String 모양 #aabbcc

custom event

  • change : 색 변경 이벤트
  • confirm : 색 선택 이벤트
  • cancel : 색 선택 취소 이벤트
